## 研究信息

### 研究背景
随着互联网技术的快速发展，博客平台已成为个人分享知识、记录学习和交流技术的重要载体。传统的博客系统往往功能单一或过于复杂，难以满足个人用户对轻量级、个性化、易用的内容管理平台的需求。同时，前后端分离的现代Web开发架构已成为行业主流趋势，通过RESTful API实现前后端解耦，能够显著提升系统的可维护性和可扩展性。

### 研究目的
本课题旨在设计并实现一个功能完善、界面美观的个人博客管理系统。系统采用前后端分离架构，后端基于Spring Boot框架提供RESTful API接口，前端使用React技术栈构建交互式用户界面，实现用户认证、文章管理、分类标签、评论互动、内容审核以及后台管理等核心功能，为个人用户提供一站式的博客内容创作与管理平台。

### 研究方法
1. **文献研究法**：调研国内外博客系统的发展现状和主流技术方案，分析前后端分离架构的优势。
2. **模块化设计**：采用分层架构设计，将系统划分为表现层、业务逻辑层和数据访问层。
3. **原型开发**：使用敏捷开发方法，迭代实现各功能模块并进行测试验证。
4. **技术选型**：后端采用Spring Boot + MyBatis-Plus + Spring Security + JWT；前端采用React + TypeScript + Ant Design + Tiptap编辑器；数据库采用MySQL。
